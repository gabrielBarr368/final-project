  • Dr Java hints:

    • If you have several files open in Dr Java, you can compile just the file you're currently editing by hitting shift-F5 or by choosing Tools->Compile Current Document.

    • You can save the history of your commands in the interactions window by choosing Tools->History->Save Interactions History. You're given a chance to edit this history before it gets saved. Later you an execute these saved interaction commands by choosing Tools->History->Execute Interactions History. By doing this you can create a test script, save it, and continually re-run it as you test your code.

About Comp 17200

Principles of Computer Science II is about problem solving with computers. It's about tackling real problems and creating real computer-based solutions. It is about logic and eloquence, efficiency and beauty, detail and abstraction. It's about problem decomposition, program design, and solution implementation. It is science and engineering. But most of all, it's about the fun that is computer science.
